# Week 4 - HTML Basics

Two beginner HTML assignments building simple webpages from scratch.

## Assignment 1 - Basic HTML Page
A simple webpage using fundamental HTML tags — headings, paragraphs, line breaks, anchor links, and unordered lists.

```html
<h1>, <p>, <br>, <a href="...">, <ul>, <li>
```

## Assignment 2 - Cat Profile Page
A structured profile page for a cat using headings, paragraphs, an image, ordered and unordered lists, a horizontal rule, and hyperlinks.

```html
<h2>, <h3>, <img src="..." height="300px" width="300px">, <ol>, <ul>, <hr>
```
