Week 4 - HTML Fundamentals
Two beginner HTML assignments building simple webpages from scratch.
Assignment 1 - Basic Webpage
Covers the basic HTML document structure. Uses headings, paragraphs, line breaks, an external hyperlink, and an unordered list.
Tags used: DOCTYPE, html, head, body, meta, h1, p, br, a, ul, li
Assignment 2 - Cat Profile Page
A structured profile page for a cat named Pochi. Includes a heading, paragraph, an image aligned to the right, an ordered list for profile details, an unordered list with links, a horizontal rule, and a footer note.
Tags used: h2, h3, p, img, ol, ul, li, a, hr, br
Concepts Covered

HTML document boilerplate and meta tags
Text formatting with headings and paragraphs
Hyperlinks and external navigation
Ordered and unordered lists
Embedding images with alignment and size attributes
Page dividers and spacing
