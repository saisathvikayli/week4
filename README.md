Week 4 - HTML Fundamentals
This repository contains two beginner HTML assignments focused on building simple webpages from scratch using core HTML tags and document structure.

Assignment 1 — Basic Webpage (assignment1.html)
A simple introductory webpage that demonstrates the most fundamental HTML tags and structure.
What it contains:

A standard HTML boilerplate with <!DOCTYPE html>, <html>, <head>, and <body>
<meta charset> and <meta viewport> tags for character encoding and responsive scaling
Paragraphs using <p>, headings using <h1>, and line breaks using <br>
A hyperlink using <a href=""> pointing to an external URL
An unordered list using <ul> and <li>

<h1>WEB pages are exciting</h1>
<a href="http://google.com">link to google</a>
<ul>
    <li>one</li>
    <li>two</li>
</ul>

Assignment 2 — Cat Profile Page (assignment2.html)
A structured profile page for a cat named "Pochi", using a wider range of HTML tags to build a more content-rich layout.
What it contains:

Multiple heading levels: <h2> for the page title, <h3> for section headers
A descriptive paragraph using <p>
An inline image using <img> with align, height, and width attributes
An ordered list <ol> for profile details (favourite food, toy, habits)
An unordered list <ul> with anchor links for external references
A horizontal rule <hr> used as a page divider
A footer-style paragraph with a last-updated note

<h2>Pochi the cat</h2>
<img align="right" src="../assets/cat.jfif" height="300px" width="300px">
<ol>
    <li>Favourite food - Tuna</li>
    <li>Favourite toy - yarn</li>
</ol>
<hr>
<p>last updated on 14th march by Pochi The Cat.</p>

Concepts Covered

HTML document structure: <!DOCTYPE>, <html>, <head>, <body>
Meta tags: charset, viewport
Text tags: <h1> through <h3>, <p>, <br>, <hr>
Links: <a href=""> for external navigation
Lists: <ul> (unordered) and <ol> (ordered) with <li>
Images: <img> with src, align, height, width attributes
Nesting elements and basic page layout without CSS
