# Week 4 - HTML Fundamentals

This repository contains two beginner HTML assignments focused on building simple webpages from scratch using core HTML tags and document structure.

---

## Assignment 1 — Basic Webpage

A simple introductory webpage that demonstrates the most fundamental HTML tags and structure.

**Document Setup**
Every HTML file starts with a standard boilerplate — `DOCTYPE` declaration, `html`, `head`, and `body` tags. The `head` contains two meta tags: one for character encoding (UTF-8) and one for viewport scaling to support different screen sizes.

**Content Used**
- A paragraph of text introducing the page
- An `h1` heading
- A second paragraph of body text
- A line break using `br`
- An anchor tag linking to google.com with visible link text
- An unordered list with five items (one through five)

**Tags covered:** `DOCTYPE`, `html`, `head`, `body`, `meta`, `h1`, `p`, `br`, `a`, `ul`, `li`

---

## Assignment 2 — Cat Profile Page

A more structured and content-rich profile page built around a fictional cat named Pochi. This assignment introduces more tags and a clearer page layout with distinct sections.

**Page Structure**
The page is divided into four sections using `h3` subheadings — Introduction, Profile, Links, and a footer note. An `h2` tag is used for the main page title.

**Content Used**
- An `h2` heading for the cat's name and `h3` tags for each section
- A paragraph with a multi-sentence description of the cat
- An image tag with `src`, `align="right"`, `height`, and `width` attributes to embed and position a photo
- An ordered list for the cat's profile (favourite food, favourite toy, sleeping habits)
- An unordered list with two anchor links pointing to external sites
- Multiple `br` tags for vertical spacing
- An `hr` horizontal rule to visually separate the footer
- A closing paragraph with a last-updated note

**Tags covered:** `h2`, `h3`, `p`, `img`, `ol`, `ul`, `li`, `a`, `hr`, `br`

---

## Concepts Covered

- HTML document structure and boilerplate setup
- Meta tags for charset and viewport
- Heading hierarchy with `h1`, `h2`, and `h3`
- Text content using paragraphs and line breaks
- Hyperlinks for external navigation
- Unordered and ordered lists
- Embedding images with positioning and size attributes
- Using horizontal rules as visual dividers
- Building a multi-section page layout using only HTML
